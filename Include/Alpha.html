Alpha

IOS Development | User Interface Design

Alpha is a free IOS App coming soon the appstore, that generates a random fortunes, words and quotes. Perfect for getting quick insperation on the go or learn some new words.

Alpha containts 150+ quotes, 50+ fortunes and holds over a 100+ words with a short description.

Tools Used

Tools used to for this project

  • Sketch - used to design the UI and all Icons for this project

  • Excel - to keep track of all the quotes and fortunes, also to make sure that there are no duplicates

  • Github - used to keep track of the progress and to keep everything up to date
