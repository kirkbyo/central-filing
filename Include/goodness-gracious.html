Goodness Gracious Sans-Serif Font

Typeface Design | Typography

Goodness Gracious is designed to be that perfect headline font. The goal for this font was to create a thin octagonal typeface that can be used for headlines or short paragraphs.

It currently contains 70+ characters ranging from the entire uppercase & lowecase alphabet, with a more then enough punctions for all your needs.

Tools Used

Tools used to accomplish this font

  • Pen and Paper - The design process for this font startted with some sketching to get a feel of some of the characters.

  • Photoshop - is where all of the character where designed. This process was very time consuming because I made sure that it was the closet thing to pixel perfect and I kept a consitency threw out the entire font.

  • Glyphs Mini - used to import and trace all the letters and symbols. All of the final touches and export to OTF was done in Glyphs.
